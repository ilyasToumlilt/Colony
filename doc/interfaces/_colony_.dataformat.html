Interface DataFormat

A DataFormat tells colony how JavaScript values should be stored in the database (in Sets, Registers, Maps). A DataFormat has to implement two functions jsToBinary and binaryToJs to convert between binary data and JavaScript values.

The default implementation is MsgpackDataFormat.
